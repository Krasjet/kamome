karasu

For editor related guides, see kamome.

karasu (カラス) is a self-hosted remote communication system designed for mathematicians. It aims to be an enhancement/replacement for email communications, which has almost non-existent support for math environments, diagrams, and typographyI might also incidentally made a platform for myself to write expository notes.

The source code can be found on GitHub. It’s mostly written in Haskell.

3 Custom Markdown commands

The following is some custom Markdown commands for myself.

  • Small caps: [dual]{.smcp} will be rendered as dualIn fact, any capital letter on this page is small caps, but this command will add extra spacing between letters.
  • Margin notes: [txt]{.mgnote} will render txt as margin notes. It will be collapsed into inline notes if your screen is not wide enough.
  • No wrap: [txt]{.nowrap} will prevent line wrapping inside txt.
